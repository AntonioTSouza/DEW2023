Tag de imagem

A tag para inserir imagem em um site HTML é "img" e ela funciona junto com o atributo "src"(source)

inserindo uma imagem local

usaremos essa imagem de exemplo

Para que essa imagem seja adicionada, o código para a imagem será o seguinte: "img src="http://localhost/DEW2023/imagens/bing%20chilling.jpg" alt="John Cena""
